# TYPE DETECTOR

`typeof()` made the right way.

## Installation

`npm install type_detector`

## Usage

```js
var typeOf = require('type_detector');

typeOf([1, 2, 3]); // returns => 'Array'
typeOf({}); // returns => 'Object'

...
```
